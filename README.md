# AWS Junior/L1 Cloud Engineer Technical Assessment

> This repository contains the completed technical assessment for the Junior Cloud Engineer role. The project demonstrates proficiency in core AWS services, automation through scripting, and Infrastructure-as-Code (IaC) principles.

The repository is organized by task areas for easy navigation:

* **`/ec2`**: AWS Launch Templates, Security Group configurations, and connectivity verification.
* **`/s3`**: S3 static website configuration and public access policies.
* **`/ebs`**: Documentation and logs for EBS volume attachment and online filesystem expansion.
* **`/iac`**: Terraform configuration files for automated resource provisioning.
* **`/scripts`**: Bash scripts for web server automation and log rotation (including crontab).
* **`/troubleshooting`**: Root-cause analysis and resolution steps for common AWS issues.
